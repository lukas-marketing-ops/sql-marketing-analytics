# SQL Marketing Analytics

SQL queries for B2B marketing analytics — built to answer real business questions
about pipeline performance, lead source quality, and deal conversion.

## Business Questions I Answer

- Which lead source has the highest win rate?
- What is the average deal value by channel?
- How does our marketing funnel convert from Lead to Closed Won?
- Which channels generate the most revenue?

## What I Learned

SQL is not about memorizing syntax. It’s about knowing which question
to ask — and translating it into data logic.

Example: LinkedIn has a 100% win rate in this dataset vs. 0% for Cold Outreach.
That’s not a code result — that’s a budget recommendation.
